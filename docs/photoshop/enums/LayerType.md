# Enumeration: LayerType

the layer type

* layer - a pixel layer
* textLayer - a text layer
* adjustmentLayer - an adjustment layer
* layerSection - a grouping layer
* smartObject - a smart object
* backgroundLayer - a background layer
* fillLayer - a fill layer
